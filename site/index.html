Para devs

VS Code + daemon local, tudo como foco produtivo.

Ative a integração opcional: Saul soma sessões do editor como tempo produtivo sem sair do localhost.

01

Ligue nas opções

Gere a chave de pareamento, ative a integração e configure a URL local. Nada sai do seu navegador.

02

Daemon local

Rode PAIRING_KEY=<chave> PORT=3123 node saul-daemon/index.cjs. O resumo diário vive em /v1/tracking/vscode/summary.

03

Extensão VS Code

Comando “Saul Goodman: preparar comando do SaulDaemon” preenche o terminal com chave/porta e envia batimentos do editor para o Chrome.
