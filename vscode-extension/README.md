# Saul Goodman — VS Code Extension

Extensão que coleta atividade no editor e envia para o Saul Daemon local.

## Como usar

1. Inicie o daemon (`saul-daemon/index.cjs`) com `PAIRING_KEY`.
2. Configure em **Settings > Saul Goodman**:
   - `saulGoodman.apiBase`
   - `saulGoodman.pairingKey`
   - `saulGoodman.enableTracking`
