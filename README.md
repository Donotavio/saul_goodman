# Saul Goodman — Monorepo

Monorepo do produto Saul Goodman: extensão Chrome, extensão VS Code, Saul Daemon e site/blog estáticos.

## Componentes

- **Extensão Chrome**: código em `src/` — entry em `chrome-extension/README.md`.
- **Saul Daemon**: `saul-daemon/README.md`.
- **Extensão VS Code**: `vscode-extension/README.md`.
- **Site/Blog**: `site/` (guia em `docs/blog.md`).

## Documentação oficial

Toda a documentação vive em `docs/`. Comece por `docs/README.md`.

## Comandos rápidos

```bash
npm install
npm run build
npm test
```
